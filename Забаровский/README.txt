Бот запускается командой из папки Bot

python messanger.py 

Команды:
Привет - чтобы начать диалог.
Помощь - чтобы узнать что умеет бот.
Как дела - чтобы узнать как себя чувствует бот.
Арифмитические операции: бот может выполнять простые арифмитические операции, к примеру 2 + 2, 2 плюс 2 и т.д.
Дата, время - показывает точные дату и время.
Анекдот - бот расскажет анекдот (1 случайный из 3-х).
Пока - прощание.

История сообщений пишется в файл textEdit_toHtml.html и загружается из него при старте новой сессии (в кодировке utf-8).
